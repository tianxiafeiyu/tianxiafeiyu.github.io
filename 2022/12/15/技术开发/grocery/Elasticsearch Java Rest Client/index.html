Elasticsearch Java Rest Client

基于 Elasticsearch 6.x

概述

Rest client 分成两部分：

  • Java Low Level REST Client
    官方低级别 es 客户端，使用 http 协议与 Elastiicsearch 集群通信，与所有 es 版本兼容。
  • Java High level REST Client
    官方高级别 es 客户端，基于低级别的客户端，它会暴露 API 特定的方法。
